# Welcome to THOX.ai

THOX.ai is building a complete ecosystem of local-first AI hardware, software, and developer tools.
Our mission is simple:

> **Powerful AI should belong to the people using it—not to distant cloud providers.**

Everything we build is focused on:

- Privacy-first AI
- Local inference
- Edge computing
- Agentic automation
- Open developer tooling
- Cross-platform deployment
- Offline-capable intelligence
