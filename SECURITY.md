# Security Policy

We take security and privacy seriously across all THOX.ai repositories.

## Reporting a vulnerability

Please do **not** open public issues for security vulnerabilities.
Instead, email details to security@thox.ai.

Include:

- Affected repository and version/commit
- Steps to reproduce
- Impact assessment
- Suggested remediation (if known)

We will acknowledge reports as quickly as possible and coordinate a responsible disclosure process.
