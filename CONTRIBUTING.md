# Contributing to THOX.ai

Thanks for your interest in contributing.

## How to contribute

1. Open an issue to discuss bugs, improvements, or features.
2. Fork and create a branch for your change.
3. Keep changes focused and well documented.
4. Open a pull request using the provided template.

## Pull request expectations

- Explain the motivation and scope.
- Include tests when practical.
- Keep commits clear and atomic.
- Be responsive to review feedback.

## Development standards

- Prefer secure, privacy-preserving defaults.
- Keep local-first and offline-capable behavior in mind.
- Avoid introducing breaking changes without discussion.
